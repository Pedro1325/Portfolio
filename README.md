# 🚀 Portfólio Dev - Pedro Henrique

Este é um projeto de portfólio desenvolvido com **React**, **Vite** e **Tailwind CSS**.

## 📋 Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:
*   [Node.js](https://nodejs.org/en/) (recomenda-se a versão LTS)
*   Um gerenciador de pacotes (npm ou yarn)

## 🔧 Instalação e Execução

Siga os passos abaixo para rodar o projeto localmente:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/pedrohssouza/portfolio.git
    ```

2.  **Acesse a pasta do projeto:**
    ```bash
    cd portfolio
    ```

3.  **Instale as dependências:**
    ```bash
    npm install
    ```

4.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

O projeto estará disponível no seu navegador em: `http://localhost:5173`

## 🏗️ Build para Produção

Para gerar a versão final para deploy, execute:
```bash
npm run build
```
Os arquivos otimizados serão gerados na pasta `/dist`.

## 🛠️ Tecnologias Principais

*   **React 18**
*   **Vite** (Build Tool)
*   **Tailwind CSS** (Estilização)
*   **Framer Motion** (Animações)
*   **Lucide React** (Ícones)
*   **Radix UI** (Componentes Acessíveis)
